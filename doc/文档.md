# HtmlHub 数据库设计文档

## 一、公共基础字段
所有数据表继承 `BaseModel`，包含通用字段：
| 字段名 | 类型 | 约束 | 说明 |
| ---- | ---- | ---- | ---- |
| id | uint | 主键、自增 | 数据唯一标识 |
| created_at | timestamp | 默认当前时间 | 创建时间 |
| updated_at | timestamp | 默认当前时间 | 更新时间 |
| deleted_at | timestamp | 索引 | 软删除标识 |

## 一、数据表结构
### 1. user（用户表）
存储用户注册、登录信息
| 字段名 | 类型 | 约束 | 索引 | 说明 |
| ---- | ---- | ---- | ---- | ---- |
| nickname | varchar(50) | NOT NULL | 唯一索引 | 用户昵称 |
| email | varchar(100) | NOT NULL | 唯一索引 | 登录邮箱 |
| password | varchar(100) | NOT NULL | - | BCrypt 加密密码 |
| uuid | char(36) | NOT NULL | 唯一索引 | 用户唯一标识 |

### 2. html_record（HTML上传记录表）
存储用户上传的HTML页面信息
| 字段名 | 类型 | 约束 | 索引 | 说明 |
| ---- | ---- | ---- | ---- | ---- |
| user_id | uint | NOT NULL | 普通索引 | 关联用户ID |
| subdomain | varchar(100) | NOT NULL | 唯一索引 | 子域名前缀（全局唯一） |
| file_name | varchar(255) | NOT NULL | - | 文件名 |
| description | varchar(500) | NULL | - | 页面简介 |
| file_size | bigint | NOT NULL, DEFAULT 0 | - | 文件大小(字节) |
| html_content | longtext | NOT NULL | - | HTML完整内容 |
| is_approved | tinyint | NOT NULL, DEFAULT 0 | - | 审核状态 |

### 3. html_record_data（HTML同步数据表）
存储页面 localStorage 云端同步数据
| 字段名 | 类型 | 约束 | 索引 | 说明 |
| ---- | ---- | ---- | ---- | ---- |
| user_id | uint | NOT NULL | 联合唯一索引 | 关联用户ID |
| html_record_id | uint | NOT NULL | 联合唯一索引 | 关联HTML记录ID |
| data_json | longtext | NOT NULL | - | 同步数据JSON字符串 |

### 4. html_record_like（HTML点赞记录表）
存储登录用户对上传页面的点赞；同一用户对同一页面仅一条有效记录（取消点赞为软删除，再次点赞恢复同一条）
| 字段名 | 类型 | 约束 | 索引 | 说明 |
| ---- | ---- | ---- | ---- | ---- |
| user_id | uint | NOT NULL | 联合唯一索引 `idx_user_html_like` | 点赞用户ID |
| html_record_id | uint | NOT NULL | 联合唯一索引、普通索引 | 被点赞的HTML记录ID |

## 三、表关联关系
1. **user → html_record**：一对多（一个用户可上传多个HTML）
2. **html_record → html_record_data**：一对一（一个页面对应一条同步数据）
3. **user → html_record_data**：间接一对多
4. **user → html_record_like**：一对多（一个用户可点赞多个页面）
5. **html_record → html_record_like**：一对多（一个页面可被多人点赞）
6. **user ↔ html_record（点赞）**：多对多，经 `html_record_like` 关联，每人每页最多一条有效点赞

# 边界
用户昵称唯一，邮箱唯一，子域名前缀唯一
未填写域名的话会随机生成一个，绝对不会冲突
同一用户对同一 `html_record_id` 仅允许一条有效点赞（`user_id` + `html_record_id` 联合唯一）
